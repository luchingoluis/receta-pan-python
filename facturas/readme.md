## Notas

Las cantidades están expresadas en porcentajes, donde la harina es la base (100%). El peso final de cada factura es de 90 gramos.

Para calcular las cantidades exactas, se debe multiplicar el porcentaje por la cantidad de harina utilizada y dividir entre 100.
